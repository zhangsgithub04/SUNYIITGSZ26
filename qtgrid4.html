How to use

  1. Use Obstacle mode and click cells to add/remove walls.
  2. Use Set start and Set goal modes to move start and goal.
  3. Click Train 500 until arrows form a path around obstacles.
  4. Click Run policy to let the trained Q-table control the agent.
  5. Use Reset Q-table after changing walls, start, or goal.
