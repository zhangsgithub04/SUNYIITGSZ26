Traditional Q-Table

8x8 Q-Table Grid Visualizer

Each grid cell is one state. The four numbers inside each state are Q-values for Up, Down, Left, and Right. The arrow shows the action with the highest Q-value.

Selected state: S0

Best action: Right

Greedy policy rule: choose argmax_a Q(s,a). That means: choose the action with the largest Q-value in the current state.

Controls

How to use the result

  1. Pick a start state and a goal state.
  2. Look inside each cell: top = Up, bottom = Down, left = Left, right = Right.
  3. The center arrow is the best action for that state.
  4. Follow arrows from the start cell to see the learned greedy policy.
  5. Click any cell to inspect its exact Q-values.
